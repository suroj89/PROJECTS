📊 Advanced Sales Data Analysis Dashboard

The Advanced Sales Data Analysis Dashboard is a full-stack data analytics project built using Python (Flask) and Power BI.
This project analyzes sales data and presents insights through a web-based dashboard and a professional Power BI report.
## 🚀 Features

* 📈 Total Sales & Profit KPIs
* 🌍 Region-wise Sales Analysis
* 🛒 Category-wise Profit Analysis
* 📅 Monthly Sales Trends
* 🏆 Top Performing Products
* 🔥 Interactive Power BI Dashboard
* 🎨 Clean and responsive UI

---

## 🛠️ Technologies Used

* **Python (Flask)** – Backend
* **Pandas** – Data analysis
* **NumPy** – Data processing
* **HTML, CSS, JavaScript** – Frontend
* **Power BI** – Data visualization & BI
